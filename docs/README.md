# Docs 目录

本目录包含EVM项目的过程文档和变更记录。

## 文件列表

| 文件 | 说明 |
|------|------|
| CHANGELOG.md | 版本变更历史 |
| NEST_IMPORT.md | 嵌套JSON导入功能说明 |

## 文档说明

### CHANGELOG.md
记录所有版本的变更历史，包括：
- 新增功能
- Bug修复
- 文档更新
- 测试用例添加

### NEST_IMPORT.md
详细说明 `--nest` 参数的功能：
- 功能概述
- 使用场景
- 格式要求
- 实现细节
- 限制说明

## 快速访问

```bash
# 查看版本历史
cat docs/CHANGELOG.md

# 查看嵌套导入功能
cat docs/NEST_IMPORT.md
```

## 维护指南

### 添加新文档
将新的过程文档或变更记录添加到此目录：
1. 创建Markdown格式的文档
2. 简明描述内容
3. 添加日期和版本号

### 文档命名规范

- 使用英文文件名：`CHANGELOG.md`（而非`变更日志.md`）
- 使用有意义的名称：`NEST_IMPORT.md` 而非`嵌套导入说明.md`
- 使用小写字母和下划线分隔单词
